== Installation ==

1. Unpack archive with plugin
2. Rename the folder to /rsscategories/
3. Copy the entire /rsscategories/ directory into your /plugins/ directory.
4. Open /plugins/rsscategories/config/rsscategories.php and set the appropriate settings
5. Activate the plugin.
